Lokální testování?

Pokud se CMS nenačítá, ujistěte se že:

  • Máte spuštěný lokální HTTP server (ne file://)
  • CMS funguje pouze online nebo s lokálním proxy serverem
  • Pro lokální testování použijte: npx decap-server
